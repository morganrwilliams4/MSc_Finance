# MSc Finance Coursework Portfolio

**University of Bath, 2025**

This repository contains a collection of coursework projects completed during my MSc in Finance program. The projects demonstrate practical applications of quantitative finance techniques, computational methods, and empirical analysis across various areas of finance.

## Key Areas Covered

- Quantitative Finance & Financial Modeling
- Trading Strategy Development & Backtesting
- Risk Analysis & Management
- Financial Econometrics & Time Series Analysis

## Technical Skills

- **Programming:** Python (pandas, numpy, statsmodels, matplotlib, scipy)
- **Tools:** Excel, WRDS, Financial Databases
- **Methods:** Statistical analysis, algorithmic trading, portfolio optimization, risk metrics

## Repository Structure

Each folder contains a specific coursework project with:
- Detailed documentation and methodology
- Well-commented source code
- Analysis and findings
